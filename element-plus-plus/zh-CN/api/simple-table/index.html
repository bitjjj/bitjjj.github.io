SimpleTable 简单表格 ​

SimpleTable 是一款非常轻量的表格组件，只使用HTML table实现所有功能。

TIP

组件需要包裹 <client-only></client-only> 标签，当用在 SSR (eg: Nuxt) 和 SSG (eg: VitePress)。

WARNING

Vitepress默认主题对组件样式有一定干扰，但不影响功能。
